# techpit-match

## サイト概要
マッチングしたユーザー間でチャットすることができるアプリ

### サイトテーマ
WebアプリケーションフレームワークであるRuby on Railsを使ってTinder風簡易マッチングアプリを作成します。

### テーマを選んだ理由
- Ruby on Railsの基礎学習としてCRUDアプリケーションの制作を行う。

### 機能
- トップページ
- ユーザー登録(Gem: devise)
- プロフィールページ/編集
